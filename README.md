🐺 Fluid Cursor 3D

Projeto desenvolvido com React + Three.js utilizando React Three Fiber para criar uma experiência visual interativa com modelo 3D, efeitos fluidos e renderização moderna em WebGL.

---

## ⚙️ Stack

- React
- Vite
- Three.js
- React Three Fiber
- Drei
- Postprocessing

---

## ⚡ Features

- GLTF 3D Rendering
- HDR Environment
- Fluid Distortion
- Perspective Camera
- Real-time WebGL
- Modular Architecture

---
